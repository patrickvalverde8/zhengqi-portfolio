# CV图片说明

## 📁 文件要求

### CV预览图片
- **文件名**: `cv-preview.jpg`
- **格式**: JPG
- **建议尺寸**: 1200x1600px (3:4比例，类似A4纸张比例)
- **文件大小**: 建议小于 2MB
- **内容**: 您的简历的JPG图片版本

## 🔄 如何创建CV图片

### 方法1: 从PDF转换
1. 打开您的PDF简历
2. 截图或导出为高质量JPG图片
3. 确保文字清晰可读
4. 重命名为 `cv-preview.jpg`

### 方法2: 在线转换工具
- 使用 PDF to JPG 在线转换工具
- 选择高质量输出
- 下载并重命名文件

### 方法3: 图片编辑软件
- 使用 Photoshop、GIMP 等软件
- 导入PDF或重新设计简历
- 导出为高质量JPG

## 📋 显示效果

- 图片会在网站CV部分完整显示
- 自动适配不同屏幕尺寸
- 用户可以点击"Download CV as PDF"下载完整PDF版本

## 📝 注意事项

- 确保图片中的文字清晰可读
- 保持专业的视觉效果
- 图片应该包含完整的简历内容
- 如果简历多页，可以制作长图或选择最重要的页面
